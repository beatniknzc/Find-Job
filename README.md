# 计算机网络
## TCP\UDP的区别
TCP和UDP是OSI模型中的运输层中的协议。TCP提供可靠的通信传输，而UDP则常被用于让广播和细节控制交给应用的通信传输。<br>
两者的区别大致如下：<br>
TCP面向连接，UDP面向非连接即发送数据前不需要建立链接<br>
TCP提供可靠的服务（数据传输），UDP无法保证<br>
TCP面向字节流，UDP面向报文<br>
TCP数据传输慢，UDP数据传输快<br>
